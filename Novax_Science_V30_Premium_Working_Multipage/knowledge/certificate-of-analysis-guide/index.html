How to Read a Certificate of Analysis

A practical guide to batch identity, analytical methods, reported purity and documentation review.

What a COA represents

A Certificate of Analysis is a batch-specific document summarising analytical information for a material. It may include identity, purity, test method, batch reference and analysis date.

Match the exact batch

Compare the product and batch identifiers on the certificate with the labelled material. A certificate from a different batch should not be substituted.

Review the method and result

Read the reported result together with the analytical method and any stated limitations. A percentage alone is not a complete description of a material.
