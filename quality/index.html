What the verification system does

The system checks an entered code against a controlled database and returns the matching record where one exists. This supports traceability but does not replace internal receiving checks, laboratory analysis or research controls.

Certificate of Analysis

A Certificate of Analysis is a batch-specific analytical document. Researchers should confirm that the product name and batch reference match the labelled material before relying on the document.

Responsible recordkeeping

Retain verification information alongside receiving records, storage records and research documentation. Use consistent batch identifiers throughout the chain of custody.
